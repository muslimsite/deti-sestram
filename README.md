# Handoff: Лендинг курса «Воспитание детей в Исламе 5.0» (Soliha)

## Overview
Одностраничный продающий лендинг курса для мусульманских мам «Воспитание детей в Исламе 5.0» от автора Солиха умм Муса. Цель страницы — рассказать о курсе, показать программу и отзывы, и привести пользователя к покупке одного из трёх тарифов (полная оплата или шариатская рассрочка) либо к контакту с менеджером.

## About the Design Files
Файлы в этом бандле — **дизайн-референс, созданный в HTML** (интерактивный прототип, показывающий задуманный вид и поведение), а **не продакшн-код для копирования 1:1**. Задача — **воссоздать этот дизайн в вашей кодовой базе** её средствами и паттернами (React/Vue/Next/SwiftUI/нативно и т. д.). Если окружения ещё нет — выберите подходящий фреймворк (рекомендуется React + Vite или Next.js) и реализуйте дизайн там.

Прототип написан как «Design Component» (single-file HTML с инлайновыми стилями) поверх дизайн-системы Soliha. При переносе используйте свою систему компонентов; ниже подробно описаны все токены, компоненты и поведение.

## Fidelity
**High-fidelity (hifi).** Финальные цвета, типографика, отступы, анимации и копирайтинг. Верстать по возможности пиксель-в-пиксель, используя библиотеки и паттерны вашего проекта. Весь текст на странице — финальный (см. `Воспитание детей в Исламе.dc.html`).

## Interactions & Behavior
- **Nav-ссылки** — плавный скролл к якорям (`html { scroll-behavior: smooth }`).
- **Аккордеоны** (программа, отзывы) — одно активное состояние (`openModule` / `openReview` в state; `-1` = всё свёрнуто для программы, `0` = первая открыта для отзывов). Клик по той же строке — свернуть.
- **Лайтбокс отзывов** — state `lightbox` (src или null); клик по картинке открывает, клик по фону/крестику закрывает.
- **Scroll-reveal**: элементы (заголовки секций, карточки, строки модулей/отзывов, тарифы, колонки сравнения, автор, финал) появляются fade + подъём на 36px при въезде во вьюпорт, со стаггером внутри группы (задержка `min(idx,6)*0.08s`). Реализовано через `IntersectionObserver` (threshold .12, rootMargin `0px 0px -6% 0px`), навешивающий класс `.is-in`. Easing `cubic-bezier(.16,1,.3,1)`, длительность .85s. Учитывать `prefers-reduced-motion: reduce` — при нём всё показывать сразу без анимации.
- **Hover кнопок** (из DS Button): подъём на 2px, у primary усиливается золотая тень, у dark/secondary — лёгкая золотая заливка `rgba(199,168,107,.12)`.
- **Responsive**: все сетки `auto-fit/minmax` → корректно перестраиваются в 1 колонку на мобильном. Клампы в размерах шрифта/паддингах.

## State Management
- `openReview: number` — индекс открытой рубрики отзывов (по умолч. 0).
- `openModule: number` — индекс открытого модуля программы (по умолч. -1).
- `lightbox: string | null` — src открытого в лайтбоксе изображения.
- Прочее — статические данные (массивы `audience`, `programData`, `tariffsBase`, `authorBio`, списки отзывов). Никаких сетевых запросов.

## Design Tokens
Полные значения — в `tokens/*.css` (скопированы в бандл). Ключевые:
- **Цвета**: золото `--gold-500 #C7A86B`, `--gold-600` (тёмнее), градиент `--gold-gradient: linear-gradient(#E3D2A8 → #C7A86B → #A77E3F)`; тёмные фоны navy `#0E1626`, near-black `#0D0D0D`, `#0A0F1C`, `--ink-900`; светлые `--cream-50 #FAF6EE`, `--cream-100 #F2EBDD`, `--white`; текст `--text-on-light #1B2A3A`, `--text-on-light-muted`, `--text-on-dark #F6F1E7`, `--text-on-dark-muted`; линии `--line-on-dark`, `--line-on-light`, `--line-gold`, `--line-gold-strong`.
- **Типографика**: display — **Playfair Display** (`--font-display`), body/ui — **Montserrat** (`--font-body`), script — **Allura** (`--font-script`); имя автора — **Hamiltone** (кастомный `.otf`, см. Assets). Body ≥16px, line-height ~1.6. Eyebrow/pill — UPPERCASE, letter-spacing .1–.2em.
- **Скругления**: `--radius-lg` ~22px (карточки/кнопки), `--radius-md`, `--radius-pill` (пилюли/бейджи — fully round).
- **Тени**: `--shadow-sm/md/lg` (тёплые, navy-tint), `--shadow-dark` (глубокая на тёмном), `--shadow-gold` (золотое свечение для primary CTA и бейджей).
- **Отступы**: секции `clamp(4rem, 9vw, 7.5rem)` по вертикали, `clamp(20px,5vw,64px)` по горизонтали; ритм 8px; max-width контента ~1080–1200px.
- **Анимация**: ease-out `cubic-bezier(.16,1,.3,1)`, 0.32–0.85s; scroll-reveal fade + translateY(36px) со стаггером.
